# Gestor PQRSF - Colegio Calasanz Suba

Sistema modernizado para la gestión de Peticiones, Quejas, Reclamos, Sugerencias y Felicitaciones (PQRSF) como aplicación satélite integrada al SGE institucional.

## 🚀 Características

- **Diseño Premium**: Interfaz moderna con React + Tailwind CSS.
- **Arquitectura Modular**: Organizado por dominios para escalabilidad.
- **Integración API**: Consumo de servicios REST del SGE.
- **Dashboard de Métricas**: Visualización en tiempo real con Recharts/Chart.js.
- **Flujos de Trabajo**: Gestión completa desde la creación hasta el cierre de solicitudes.

## 🛠️ Tecnologías

- **Frontend**: React 18, Vite, Framer Motion.
- **Estilos**: Tailwind CSS.
- **Validación**: React Hook Form + Zod.
- **Comunicación**: Axios con interceptores JWT.

## 📦 Instalación

1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar entorno de desarrollo:
   ```bash
   npm run dev
   ```
